## Tehnologii folosite
- React + Vite + TypeScript
- Tailwind CSS
- Strapi 5 CMS (Strapi Cloud)
- Netlify (hosting frontend)
- 3 fonturi web: Outfit, Playfair Display, Geist Mono

## Structura proiectului
- `src/components` - componente reutilizabile (Navbar, Footer, Cards)
- `src/pages` - paginile site-ului (Home, Articole, Categorii, Despre, Contact)
- `src/services` - servicii API pentru comunicarea cu Strapi
- `src/hooks` - custom hooks pentru fetch date
- `src/types` - tipuri TypeScript
- `src/utils` - utilitare (imagini, formatare)
